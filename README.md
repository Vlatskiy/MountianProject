## Сайт портфолио

#### В данном проекте использовались следующие технологии:
* Workflow : Webpack, Gulp, npm
* HTML5
* CSS3
* JavaScript, JQuery, Node.js
* Sass, Pug
* Работа с векторной графикой. Создание и использование спрайтов
* Google Maps API

#### Реализованы:
* Адаптивная, кроссбраузерная вёрстка
* Прелоадер
* Параллакс эффекты
* Слайдер
* Google Maps API
* CSS animations
